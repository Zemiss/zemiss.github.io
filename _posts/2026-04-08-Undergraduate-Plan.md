# 本科计划书 (2026.04 - 2028.06)

## 1. 核心愿景与战略目标

**核心愿景**：以**虚拟细胞甲基化数据扩增**为研究切入点，通过高性能算法实现与高质量学术产出，构建涵盖“高绩点、顶刊论文、竞赛实战、工业实习”的全方位竞争优势，冲刺顶尖学府的研究生席位。

**核心指标 (KPIs)**：
*   **学术产出**：发表 1 篇 CCF-A 类期刊/会议论文（聚焦 AI + 生物信息交叉方向）。
*   **学业表现**：保持 GPA 3.7/4.0 以上，夯实数学与计算机底层基础。
*   **竞赛实战**：获得 Kaggle 生物/表观遗传相关赛事 Top 10% 或金牌。
*   **职业进阶**：完成 1-2 段 LLM 数据增强相关的头部互联网企业算法岗实习。
*   **语言门槛**：雅思 (IELTS) 稳定在 7.0 分以上。

---

## 2. 研究方向与技术路径

**研究课题**：基于深度学习的单细胞 DNA 甲基化数据扩增与修复。

**技术逻辑**：
针对单细胞甲基化数据的高稀疏性与强噪声特征，利用 **PyTorch** 构建基于 **Transformer** 或 **扩散模型** 的生成式架构，实现高质量虚拟细胞数据的生成，为下游表观遗传学分析提供数据支撑。

**预期成果**：
*   开发一套针对单细胞组学数据的增强算子。
*   验证扩增数据在细胞分型、聚类分析等任务中的增益效果。
*   目标期刊：*Bioinformatics*、*Briefings in Bioinformatics* 或 CCF-A 类人工智能期刊。

## 3. 阶段性执行路线图

### 第一阶段：夯实基础与实验启动 (2026.04 – 2026.08)
*   **学业**：确保本学期核心课绩点。
*   **科研**：完成《单细胞甲基化数据增强》文献综述，确立 Baseline 模型。
*   **英语**：启动雅思首轮复习，侧重学术阅读与词汇积累。
*   **实习**：完善 GitHub 项目集，投递 LLM 数据方向暑期实习。

### 第二阶段：模型攻坚与数据实战 (2026.09 – 2027.02)
*   **科研**：完成模型迭代与消融实验，撰写论文初稿。
*   **竞赛**：深度参与 Kaggle 生物信息学竞赛，沉淀特征工程经验。
*   **实习**：进入 1 段全职算法实习，积累 LLM 大模型数据处理的工程实务。

### 第三阶段：成果转化与语言冲刺 (2027.03 – 2027.08)
*   **论文**：完成 CCF-A 类期刊投稿，根据评审意见进行审稿回执（Rebuttal）。
*   **语言**：参加雅思考试，确保总分 7.0，写作不低于 6.5。
*   **提升**：整理 Kaggle 竞赛成果，开源相关代码库，提升学术影响力。

### 第四阶段：研究生申请与定稿 (2027.09 – 2028.06)
*   **申请**：准备全套文书（SOP/CV），联系推荐人，投递目标院校。
*   **毕业**：完成高质量毕业论文，确保论文最终见刊或在线发表。
*   **衔接**：获取研究生录取通知，启动后续博士课题预研。
